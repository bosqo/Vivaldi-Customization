# Shortcut Cheatsheet

This cheatsheet lists the custom shortcuts configured in the `Preferences` file that differ from the Vivaldi defaults.

## Extension / Custom Commands

| Shortcut | Command | ID / Type |
| :--- | :--- | :--- |
| **Alt+Shift+G** | Toggle Tab Group | `toggle-tab-group` |
| **Ctrl+.** (Period) | Execute Side Panel | `execute_side_panel` |
| **Ctrl+Shift+1** | Custom Hotkey 1 | `hotkey_1` |
| **Ctrl+Shift+2** | Custom Hotkey 3 | `hotkey_3` |
| **Ctrl+Shift+3** | Custom Hotkey 2 | `hotkey_2` |

> [!NOTE]
> These shortcuts are defined in the `extensions.commands` section of the `Preferences` file. `hotkey_x` likely refers to mapped Command Chains or specific extension actions.
